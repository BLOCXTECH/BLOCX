# Autolykos

## Top-Level File: `AutolykosPowScheme`

### Steps to Run the AutolykosPowScheme Test Cases

1. **Navigate to the `run` directory**:
   ```sh
   cd run
   ```

2. **Run the desired test case**:
   ```sh
   make TEST="TEST_NAME"
   ```

### Example: Running a Specific Test Case

To run the `test_validate_pow_1.cpp` test from the `tests` directory, follow these steps:

1. **Navigate to the `run` directory**:
   ```sh
   cd run
   ```

2. **Execute the `test_validate_pow_1` test case**:
   ```sh
   make TEST=test_validate_pow_1
   ```

This command will run the `test_validate_pow_1` test case.
